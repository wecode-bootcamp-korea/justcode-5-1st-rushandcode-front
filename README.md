  - 상품 리스트와 연동
  
#### 손소희
- **`회원가입&로그인`**
  - 로그인/회원가입 페이지 레이아웃
  - Input 정보 오류 시 오류 별 메세지 알림 기능<br/>
   ( 최소 글자 입력조건, 영문대/소문자, 숫자, 특수문자 중 2가지 이상 조합 등)
  - 회원가입/로그인 버튼 클릭 시 조건에 맞다면 해당 화면으로 이동
- **`이벤트`**
  - 이벤트 페이지 레이아웃
  - 이벤트 클릭 시 해당 이벤트 페이지로 이동
